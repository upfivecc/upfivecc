我一开始对用人工智能做作业非常怀疑，因为感觉像作弊，但后来我意识到“帮我做作业”和“帮我真正理解”之间有着巨大的差别。以下是我如何从勉强及格到真正理解题目的提示：

## 1. “我迷路了”家庭作业助手

```text
我正在做[具体问题/作业]。这是我目前尝试过的：[你的尝试]。我觉得我卡住了，因为[你感到困惑的地方]。不要直接告诉我答案——告诉我我遗漏了什么概念，并提出一些能帮助我理解的问题。
```

## 2. “像给五岁小孩解释一样解释（但要让人记住）”

```text
“我正在学习[语言]，但总是搞错[某个语法点/概念]。我读过课本上的解释，但还是不明白。请用[我感兴趣的领域]的例子来解释一下，然后给我5个难度逐渐增加的练习句子。”
```

## 3. “考试焦虑克星”

```text
“我有一个[科目]考试，时间在[时间段]，考试内容涵盖[主题]。我最擅长[X]，但[Y]比较弱。请帮我制定一个学习计划，重点突出我的薄弱环节，包含主动回忆练习，而不是简单地让我‘复习所有内容’。我今天应该做什么？”
```

## 4. “研究兔子洞导航器”

```text
我正在研究关于[论文主题/问题]的论文。我找到了一些资料支持[观点 A]，也找到了一些资料支持[观点 B]。请帮我理解这方面的学术争论，我应该问哪些问题，以及哪些搜索词能找到比我目前找到的更好的资料。
```

## 5. “定制学习路径”

```text
我想在[时间范围]内学习[技能/科目]。我的学习风格是[视觉型/实践型/等等]，我每周有[时间投入]，并且我已经达到[当前水平]。请为我制定一个包含里程碑的每周计划，以便我能够实际跟踪学习进度。
```

## 6. “等等，这到底是什么意思？”

```text
“我在[学科]课上读[复杂概念]。课本上的解释[令人困惑的解释]。请帮我分析一下：用一句话概括核心思想？举个现实生活中的例子？最常见的误解是什么？为什么这很重要？”
```

## 7. “将我杂乱的笔记变成有用的东西”

```text
这是我关于[主题]的课堂笔记：[粘贴笔记]。笔记很乱。请帮我整理成：（1）主要概念，（2）支撑细节，（3）需要复习的内容，以及（4）可能的考题。另外，请标记出任何不完整的内容。
```

## 8. “让我深入思考”

```text
我刚学习了[概念/主题]。我理解了基本概念，但我想进行批判性思考。这个想法的局限性是什么？它基于哪些假设？它如何应用于[不同的情境]？持不同意见的人会提出什么论点？
```

关键在于： 一定要把已经尝试过或已经知道的内容都写进去。人工智能更擅长填补知识空白，而不是从零开始。说实话，解释你遇到的难题本身，很多时候就能帮你找到答案。

对我来说，改变在于： 我不再因为没能立刻理解而感到笨，而是开始把学习看作一场对话，而不是死记硬背。我的成绩提高了，因为我真正理解了学习内容，而不是找到了捷径。
