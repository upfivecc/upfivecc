大多数人使用人工智能的方式都和搜索引擎类似。他们提出问题，然后就听天由命。但只要稍微改变一下提问方式，人工智能就能从搜索引擎变成你的思考伙伴。

以下是一些我经过测试之后效果非常不错的方法。

## 让人工智能思考，而不仅仅是回答问题

以 **我们换个角度思考这个问题** 开头 ——这样可以避免千篇一律的回答，从而获得富有创意的答案，而不是泛泛而谈。

问自己 **我在这里忽略了什么？** ——这能发现你之前未曾意识到的盲点，并挑战你的固有观念。

就说 **给我解释一下** ——这句话对简单的主题也适用。“解释一下如何制作咖啡”就能让你了解其中的科学原理、技巧等等。


使用 **如果你是我，你会怎么做？** 这个问题——可以让人工智能给出实际的意见，而不是中立的建议。这样更有用。

在任何问题后面加上 **我真正想问的是** ——例如：“我怎样才能升职？我真正想问的是：我怎样才能脱颖而出而不令人讨厌？”


最后问一句 **我还需要知道什么？** ——补充一些你从未想过要问的背景信息和警告。


## 提升推理能力

添加 **逐步解释你的推理过程** 功能 ——了解人工智能如何得出答案。这将彻底改变复杂决策的解决方式。

使用 **这里有什么相反的观点？** ——打破信息茧房。人工智能会反驳自己的第一反应。

说 **假设我一无所知** ——这能让你打下基础，并发现你知识上的不足。

问问自己 **我应该问什么问题？** ——通常情况下，你提出的问题并不是正确的问题。这样做可以找到更好的问题。


使用 **先提供入门版，再提供专家版** ——一举两得。入门版阐明概念，专家版深入讲解。

最后提出 **什么会导致这种做法适得其反？** ——在制定策略之前，找出可能出现的失败模式。


## 化问题为优势

将 **……有什么问题** 替换为 **……的妙处** ——“拖延的妙处是什么？”揭示了拖延是一种创造力工具。


利用 **[问题]实际上如何让我们变得更强大？** 这个问题——每一次挣扎都变成了隐形的训练。


问问自己 **我们正在暗中培养哪些技能……** ——这能把烦人的事情变成强化训练。Zoom 疲劳也能变成能量管理训练。


尝试问 **（挑战）在哪些方面让我们为……做好准备** ——将当前的痛苦与未来的力量联系起来。


问自己 **如果[问题]消失了，我们会失去什么？** ——通常，我们憎恨的东西反而能保护我们免受更糟糕的事情发生。


最后提出问题：**这种摩擦力究竟有什么用？** ——摩擦力可以增加抓地力，锻炼肌肉，防止灾难发生。

## 让枯燥的话题变得有趣

加上 **背后隐藏着什么故事……** ——万事万物都充满阴谋。办公室政治、供应链、文件系统。

使用 **外星人类学家会注意到什么……** ——获得局外人的视角，揭示我们忽略的奇怪事物。

问 **阴谋论版本的……是什么？** ——不是真正的阴谋，而是能揭示规律的连贯思维。

试试问 **（一件看似无聊的事）怎么可能是一种生存技能？** ——从进化论的角度来看，一切都变得有意义了。

使用 **如果我们把[某件事]推向逻辑极端会发生什么？** ——将想法推向极限，揭示可能性和问题。

以 **这揭示了人性的什么？** 结尾 ——这会让一切都变得意义深远。每个话题都成了了解我们自身的窗口。


## 快速发出专家的声音


问 **初学者在哪些方面总是犯错……** ——这能让你显得像一个见过这些错误的人。

使用 **业内人士都知道的潜规则是什么？** ——揭示那些没有人写下来的内部信息。

试试 **一位拥有十年经验的老手会给新人什么建议……** ——获取久经沙场的智慧、捷径和现实检验。

问 **专家眼中显而易见，但外行人却看不到的事情……** ——揭示专业人士自动使用的思维模型。

使用 **如何通过微妙的迹象判断一个人是否真正了解[主题]？** ——这能帮助你识别和展现专业知识，并找到可信度指标。

最后提出一个问题：**关于……，没有人会告诉你的一件事是什么？** ——那就是所学与现实之间的差距。


## 挖掘隐藏的洞见

问 **……的二阶效应是什么？** ——这超越了显而易见的后果。“远程办公的二阶效应是什么？”这个问题能让你看到文化转变，而不仅仅是生产力的变化。

使用 **如果这件事很简单会是什么样子？** 这个问题 ——可以剥离复杂性，往往能揭示隐藏在层层复杂表象之下的核心解决方案。

试试 **80/20 法则** ——找出驱动大部分结果的关键因素。适用于任何主题。

问问自己 **如果有人不同意这个观点，他们会怎么说？** ——这相当于自带一个反方辩护人，可以检验你的想法是否正确。

运用 **这里蕴含的深层教训是什么？** ——超越具体建议，直指根本原则。将战术转化为战略。

最后提出 **这与其他一切有何关联？** ——描绘关系图。展示孤立的主题如何融入更大的系统。


## 产生更好的想法

先问问自己 **如果情况反过来会怎样？** ——这会颠覆你的固有观念，往往能带来突破性的洞见。

使用 **100 年后会是什么样子？** ——时间视角可以展现长期趋势和影响。

问 **如果是[名人]遇到这种情况会怎么做？** ——借鉴他人的思维方式。“如果是史蒂夫·乔布斯会怎么做？”——这能让你接触到设计思维。


试试 **……中最违反直觉的方面是什么？** ——发现大多数人忽略的令人惊讶的真相。

运用 **如果每个人都这样做会发生什么？** ——规模化思维。揭示系统性影响。

最后问问自己：**这件事最简单的版本是什么？** ——去繁琐地归纳，只保留最基本的要素。通常，最简单的版本才是正确的版本。


## 如何将这些技术结合起来使用


当你组合使用不同的提示时，奇妙的事情就会发生。不要只问一个问题，而是使用一系列问题：

```text
“我们换个角度想一想——如果你是我，你会怎么做才能升职？我忽略了什么？什么会适得其反？”
```

或者：

```text
“这个告知书的签名背后隐藏着什么故事？一位外星人类学家会注意到什么？这又揭示了人性的哪些方面？”
```

先从一种技巧入手。熟练之后，再将它们串联起来，以获得更深入的见解。


## 为什么这种方法有效


这些提示之所以有效，是因为它们改变了人工智能处理请求的方式。它不再只是检索信息，而是开始推理；它不再只是提供帮助，而是开始真正地发挥作用。


大多数人把人工智能当作一个功能强大的搜索引擎。而这些技术可以把它变成一个思考伙伴。


最好的提示不仅能帮你找到答案，还能帮你提出更好的问题。
