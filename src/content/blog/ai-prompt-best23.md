这是我今年最喜欢的学习提示词。我用它来开启任何主题的学习。它将学习过程分解成可操作的步骤，包括研究、总结和测试。它为你构建了一个框架。当然，你仍然需要完成它。



[SUBJECT] = 要学习的主题或技能
[CURRENT_LEVEL] = 当前知识水平（初级 / 中级 / 高级）
[TIME_AVAILABLE] = 每周可用于学习的时间（小时）
[LEARNING_STYLE] = 偏好的学习方式（视觉 / 听觉 / 动手实践 / 阅读）
[GOAL] = 明确的学习目标或期望达到的技能水平

⸻

Step 1：知识评估（Knowledge Assessment）
	1.	将 [SUBJECT] 拆分为核心组成部分
	2.	评估每个组成部分的复杂度层级
	3.	梳理前置知识与依赖关系
	4.	识别基础概念与关键原理

输出： 详细的技能树与学习层级结构

⸻

Step 2：学习路径设计（Learning Path Design）
	1.	基于 [CURRENT_LEVEL] 设定学习进阶里程碑
	2.	按最优学习顺序组织知识主题
	3.	预估每个主题所需的学习时间
	4.	结合 [TIME_AVAILABLE] 的时间限制进行规划

输出： 带时间规划的结构化学习路线图

⸻

Step 3：学习资源整理（Resource Curation）
	1.	根据 [LEARNING_STYLE] 筛选匹配的学习材料：
	•	视频课程
	•	书籍 / 文章
	•	互动练习
	•	实战项目
	2.	按学习效果对资源进行排序
	3.	生成学习资源播放/阅读清单

输出： 按优先级排序的完整学习资源列表

⸻

Step 4：练习体系设计（Practice Framework）
	1.	为每个主题设计配套练习
	2.	构建真实世界的应用场景
	3.	设定学习进度检查点
	4.	规划复习与巩固的时间间隔

输出： 包含间隔重复策略的练习计划

⸻

Step 5：学习进度追踪系统（Progress Tracking System）
	1.	定义可量化的学习进度指标
	2.	制定评估与测试标准
	3.	设计反馈与改进机制
	4.	建立里程碑完成度衡量方式

输出： 学习进度追踪模板与评估基准

⸻

Step 6：学习计划生成（Study Schedule Generation）
	1.	将学习内容拆解为每日 / 每周任务
	2.	合理安排休息与复习时间
	3.	设置阶段性检查与测评
	4.	平衡理论学习与实践训练

输出： 与 [TIME_AVAILABLE] 匹配的详细学习时间表
